# Mesto

Проект Mesto - интерактивная галерея с возможностью добавлять, удалять и лайкать карточки.

## Ссылка на проект

https://grozbobik.github.io/Yandex_Practicum/

## Команды

- `npm run dev` - запуск локального сервера разработки
- `npm run build` - сборка проекта
- `npm run preview` - предпросмотр собранного проекта
